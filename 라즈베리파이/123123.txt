8장
프로세스: 실행중인 프로그램

종류
-시스템프로세스 : 유닉스 운영에 필요한 기능 수행
-사용자프로세스 : 사용자들이 실행시킨 프로세스

프로세스
-데몬 : 유닉스 커널에 의해 실행되는 프로세스로 특정
	서비스 제공
-부모 : 자식프로세스를 만드는 프로세스
-자식 : 부모에의해 생성된 프로세스
	실행이 끝나면 부모 프로세스에 결과주고 
	종료
-고아 : 실행중 부모가종료된 프로세스
	고아는 1번프로세스를 새로운부모로 가진다
-좀비 : 부모가 종료처리를 하지않는 프로세스
	종료가되도 할당된 자원이 회수안된

ps 프로세스 목록보기
pgrep : 특정 프로세스 정보 검색
prstat : 동작중인 프로세스의 통계정보를 출력
top  : 주기적으로 현재 실행중인 프로세스에 대한 정보 출력
nohup : 로그아웃한 다음에도 지정한 백그라운드 작업이
	 실행완료되도록 해야 할 때 사용
